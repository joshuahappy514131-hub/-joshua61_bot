# 🤖 @joshua6_bot - Production-Ready Telegram AI Assistant

A high-performance, robust, and production-ready Telegram AI chatbot powered by **Python 3.12**, **python-telegram-bot (v21+)**, and the latest **OpenAI API**. Built with full context awareness, sliding-window conversation memory, typing indicators, and secure environment configuration.

---

## Features

- **Natural AI Dialogues:** Seamlessly powered by OpenAI models (`gpt-4o` or similar).
- **Persistent Chat Context:** Retains individual conversational memory partitioned per chat session.
- **Typing Indicator:** Automatically shows "Joshua AI is typing..." while fetching completions.
- **Markdown Formatting:** Fully supports bold text, italic text, bullet points, and code blocks.
- **Robust Error Handling:** Recovers safely from timeouts, network interruptions, and rate limits without crashing.
- **Railway & GitHub Ready:** Pre-configured for seamless cloud deployment.

---

## Commands

- `/start` - Launch introduction message
- `/help` - Show command reference guide
- `/clear` - Reset chat context and session history
- `/about` - Display bot information
- `/ping` - Check server latency and system uptime
